## Vue.js Termék szűrő példa 

A repository a [https://vuejs.hu/cikk/egyszeru-termek-szuro/31](https://vuejs.hu/cikk/egyszeru-termek-szuro/31) cikk példakódja.

### Installálás

`git clone https://github.com/vuejs-hu/product-filter.git`

`composer install`

`npm install`

`npm run dev`

### Indítás

`php artisan serve`
